
## 新词分词工具

启动方式：执行launch.py脚本

结果暂存在Chinese_segment_augment/data目录下。

执行新的任务将会删除之前记录。

## 内容和使用工具：

* 爬取的内容大致分娱乐信息流、体育新闻、国内外热点以及新兴产业电竞等。

* 爬取用的主要使用requests、beautifulsoup两个工具以及涉及到的解密和代理等。

* 分词由[中文分词新词发现](https://github.com/zhanzecheng/Chinese_segment_augment.git)完成,详细内容见下。
